# Shirt Order Form Website

## How to Deploy on GitHub Pages

1. Create a GitHub account if you don't have one.
2. Create a new repository (e.g., `shirt-order-form`).
3. Upload these files (`index.html`, `style.css`, `README.md`).
4. Go to **Settings > Pages**.
5. Under **Source**, select **Deploy from a branch**, choose `main` branch and root folder.
6. Click **Save**.
7. Your site will be live at `https://yourusername.github.io/shirt-order-form/`.

## Important
Replace `YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL` in `index.html` with your actual Google Apps Script Web App URL.
